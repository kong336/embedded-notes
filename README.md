# Embedded Notes

Official-source-first notes and resource mapping for embedded systems learning, firmware practice, and electronics competition preparation.

## Focus Areas

- STM32 / ESP32 / Arduino
- firmware basics and peripheral bring-up
- electronics competition preparation
- RTOS and project organization
- hardware tools and official SDKs

## Repository Structure

- `resources/official-resource-map.md`: official or primary-source links for embedded learning and competition preparation

## Principles

- prefer official documentation over reposts
- turn fragmented learning into reusable notes
- keep the path from beginner projects to contest projects clear

## Current Direction

- build a practical learning map for hardware and embedded work
- collect reliable starting points for development boards, SDKs, and tools
- turn study notes into small but publishable repositories
